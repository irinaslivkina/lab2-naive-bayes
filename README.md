# Lab 2 – Naive Bayes Sentiment Analysis

## Description
This project implements a Naive Bayes classifier for sentiment analysis of tweets.

## Steps
1. Data preprocessing using process_tweet
2. Building word frequency dictionary
3. Computing log prior probabilities
4. Computing log likelihoods
5. Implementing Naive Bayes classifier
6. Evaluating accuracy
7. Error analysis
8. Testing on custom tweets

## Dataset
Sentiment Polarity Dataset Version 2.0 (NLTK movie_reviews)
